# Spaced Repetition

Spaced repetition schedules reviews at increasing intervals timed to land just
before you would forget. Each successful recall flattens the forgetting curve, so
the next interval can be longer. The effect was first quantified by Hermann
Ebbinghaus in 1885 using nonsense syllables.

Active recall — retrieving an answer from memory rather than re-reading it — is
what makes the scheduling work. Recognition ("that looks familiar") strengthens
memory far less than retrieval. This is why flashcards beat highlighting: the
card forces a retrieval attempt before showing the answer.

The testing effect compounds with spacing: a review that requires effortful recall
at the edge of forgetting produces more durable memory than an easy review minutes
after learning. Practical systems (SuperMemo's SM-2, Anki) implement this with
per-card ease factors that widen or narrow intervals based on self-graded recall
quality.
